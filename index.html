¿Problemas con Firebase?

  1. Asegúrate de que Firestore esté habilitado en tu proyecto de Firebase
  2. Configura las reglas de seguridad en Firestore para permitir lectura/escritura:
    rules_version = '2';
    service cloud.firestore {
      match /databases/{database}/documents {
        match /{document=**} {
          allow read, write: if true;
        }
      }
    }
  3. Verifica que la colección "reminders" exista en Firestore

Si los problemas persisten, puedes como alternativa temporal.
